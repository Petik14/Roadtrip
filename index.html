Trasa od Niše

Niš → Preševo → Skopje → Gevgelija → Evzonoi → Soluň / Thessaloniki → Nea Moudania → Nikiti → Neos Marmaras → Toroni → Porto Koufo → Kalamitsi → Sarti → Kavourotrypes / Orange Beach → Armenistis → Platanitsi Beach → Vourvourou → Karidi Beach → Ormos Panagias → Pyrgadikia → Ierissos → Stratoni → Olympiada → Stavros → Asprovalta → Lion of Amphipolis → Promachonas / Kulata border crossing → Sofia → Niš

Co bych osobně nevynechal

  • Niš: Niš Fortress nebo Skull Tower.
  • Thessaloniki: White Tower + waterfront + večerní jídlo/bary.
  • Sithonia: Neos Marmaras, Porto Koufo, Sarti, Kavourotrypes/Orange Beach, Vourvourou, Karidi Beach.
  • Návrat mimo Soluň: Pyrgadikia, Olympiada/Ancient Stagira, Lion of Amphipolis.
  • Sofia: Alexander Nevsky + Serdica + Vitosha Boulevard.
